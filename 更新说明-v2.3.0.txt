========================================
阿旺小厨房 v2.2.4 → v2.3.0 更新说明
========================================

一、新增功能
-----------
1. 公告栏滚动
   - 管理员在设置页可编辑公告内容+启用开关
   - 客户端页面顶部自动滚动展示公告

2. 客户自助申请+审核
   - 新客户输入姓名 → 查不到 → 显示"申请加入"按钮
   - 提交后管理员在客户管理页看到"待审核"列表
   - 管理员可"通过"或"拒绝"申请

3. 递个纸条（对话消息）
   - 客户端导航栏新增"递个纸条"标签
   - 显示与管理员的历史对话，可发消息
   - 有未读回复时标签上显示红色数字气泡
   - 管理员端新增"纸条"导航页面
   - 显示所有发纸条的客户列表，带未读红圈
   - 点击客户可回复对话

4. 访问日志
   - 管理员端新增"日志"导航页面
   - 记录每位客户的登录时间（每日去重）
   - 支持搜索、分页、批量删除

二、优化改进
-----------
1. 标题栏显示当前客户姓名（👋 wt）
2. 管理员可编辑客户信息（手机号/地址/备注）
3. 管理员可删除整条对话消息
4. 管理员可删除整个客户的对话记录

三、管理员导航栏更新
-------------------
菜品 · 客户 · 订单 · 纸条 · 日志 · 设置

四、数据库变更（必须在Supabase执行）
-------------------------------------
- customers表新增：status, applied_at 字段
- settings表新增：announcement, announcement_enabled 字段
- 新增消息表：messages
- 新增日志表：access_logs
- 文件位置：supabase/migration-v2.3.0.sql
